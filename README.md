# Data-analysis

Examples of python-based scripts for data vizualisation and setting up machine learning algorithms:

- Datasets
- Seaborn
- Matplotlib
- KNN / K-means algorithms
- E-mail donwload - attached files
- Categorical, distribution matrix plots..
